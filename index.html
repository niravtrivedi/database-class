Database class are collection of various function makes data manipulation easy with database.It is very simple to configure/installation and easily embed with your code.

Below some steps for using database-class.

Installation /Configure :: Step 1: Create MySQL connection Step 2: Include Database.php file after connection Step 2: Create Object that's It :)

Config

include "config.php"; //include connection file here include_once "Database.php"; //you can define connection class in Database.php $db = new Database();

Insert Example

$post = array('field'=>'value','field1'=>'value'); $db->insert('table_name',$post); echo "Inseted Row ID is :: ".$db->InserId; echo "

Select Example

$db->select('users'); while($r=$db->fetch()){ echo $r['email_user']; echo "
"; } echo "Total Rows : ", $db->NumRows();

Select with where

You can use various conditions here such as like , regexp and etc...

$db->select('table_name','*','condition = "match_peference"'); $r = $db->fetch();

With where condition as array concate with AND,OR and Other

$whr=array('AND'=>array('f1'=>'m1','f2'=>'v2')); $db->select('table_name','*',$whr); $r=$db->fetch();

Select with different MySQL Clause (Order by , group by ,limit , start examples)

$db->order_by = 'id ASC'; $db->start = 2; $db->limit = 4; $db->group_by = 'email_user'; $db->select('users'); while($r=$db->fetch()){ echo $r['email_user']; echo "
"; } echo "Total Rows : ", $db->NumRows();

Update records

Get total rows count //without condition

echo "Total Rows : ",$db->Count('table_name');

Get total rows count by condition

echo "Total Rows : ",$db->Count('table_name','condition = "mactch"');

Row count with group by
